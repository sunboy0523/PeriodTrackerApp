# 经期助手 (PeriodTracker)

一款简洁、功能完整的月经记录安卓应用。

## 功能特性

- **日历视图**：直观显示经期、排卵期和安全期
- **智能预测**：自动预测下次月经和排卵日
- **统计分析**：图表展示周期变化趋势
- **历史管理**：查看和管理所有历史记录

## 技术栈

- **语言**：Kotlin
- **架构**：MVVM + Repository 模式
- **数据库**：Room
- **UI**：Material Design 3
- **图表**：MPAndroidChart

## 项目结构

```
app/src/main/java/com/periodtracker/
├── data/           # 数据层（实体、DAO、数据库、仓库）
├── viewmodel/      # 业务逻辑层
└── ui/             # 界面层
    └── fragments/  # 日历、统计、历史页面
```

## 如何运行

1. 使用 Android Studio 打开项目
2. 同步 Gradle 依赖
3. 连接设备或启动模拟器
4. 点击运行

## 代码生成说明

本项目为 AI 生成的示例项目，包含了完整的经期预测算法：
- 周期长度计算
- 排卵日预测（下次月经前14天）
- 排卵期预测（排卵日前5天后1天）
- 周期规律性评估
